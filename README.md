# 🐳 Containers & Docker Lab

Este repositorio reúne experimentos, ejemplos y configuraciones relacionadas con **contenedores y Docker**.  
Su objetivo es servir como un espacio de aprendizaje práctico para entender cómo funcionan los contenedores y cómo utilizarlos en distintos entornos de desarrollo.

---

## 📦 Contenido del repositorio

Aquí podrás encontrar diferentes tipos de recursos relacionados con Docker:

- Ejemplos de **Dockerfiles**
- Configuraciones de **docker-compose**
- Pruebas de despliegue de aplicaciones en contenedores
- Entornos de desarrollo reproducibles
- Integración de servicios en múltiples contenedores
- Pruebas de redes y volúmenes en Docker

---

## 🎯 Objetivos

- Aprender el funcionamiento interno de los contenedores
- Crear entornos de desarrollo portables
- Experimentar con arquitecturas basadas en contenedores
- Documentar configuraciones útiles para proyectos futuros

---

## 🛠 Tecnologías utilizadas

Entre las herramientas utilizadas en este repositorio se encuentran:

- Docker
- Docker Compose
- Linux
- Redes de contenedores
- Volúmenes persistentes
